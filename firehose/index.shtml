Firehose

You want my chat traffic? Sure, take it

An encrypted chat program designed to avoid some common pitfalls (and probably fall into some uncommon ones, since the author isn't a cryptography expert)

TL;DR: it allows you to broadcast messages to thousands of people (reducing the effectiveness of traffic analysis). Messages are encrypted with GPG, so that only the intended recipient will be able to decode them (I'm not stupid enough to roll my own encryption / message signing framework)

Public broadcasting of secret data would normally be bad, but if they're sniffing it anyway, we might as well build a system that can handle it...

  • No binary builds until I see some interest in them (there shouldn't be any, because you shouldn't trust random binaries from the internet to be free from backdoors)
